Experimental visualization and query interface of perception logs. Existing commands:

  • objects - > shows all percieved objects from the belief state
  • scenes - > shows all scenes processesed by RoboSherlock
  • views(objID) - > shows all views of an object from the belief state; e.g. objects(2)

Extending the query language will allow queries such as 'show all views of object2 where the shape annotator had value X'
